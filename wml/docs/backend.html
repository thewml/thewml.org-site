Backends

These programs provide the core functionality of WML which is always present (and only can be reduced via option WML -p under runtime). They are used in the given sequential pass order.

  • wml_p1_ipp
    Pass1: Include Preprocessor
  • wml_p2_mp4h
    Pass2: Macro Processor for HTML Documents
  • wml_p3_eperl
    Pass3: Embedded Perl 5 Language
  • wml_p4_gm4
    Pass4: GNU M4
  • wml_p5_divert
    Pass5: Divert
  • wml_p6_asubst
    Pass6: Area Subst
  • wml_p7_htmlfix
    Pass7: HTML Fix
  • wml_p8_htmlstrip
    Pass8: HTML Strip
  • wml_p9_slice
    Pass9: Slice
